1 - Outdoor ground tiles

These tiles make up 5 different ‘terrains’:

-Standard grass

-Light grass

-Dark grass

-Dirt

-Cobblestone pavement

Aside from standard grass, each terrain is a complete 2-corner wang set.

Each can make up any configuration as long as it is at least 2 width/2 height.

Light grass, dark grass, and dirt all transition to standard grass only. Pavement transitions to dirt.

These are detail tiles to spruce up the wang sets.

-The first three are what make up the standard grass tiles.

-The last four are pavement detail tiles.

-Take care as the blue and purple highlighted dirt details are larger than 1 tile (1x2, 2x2 respectively).

Tips:

-To avoid obvious repetition, try not to use the same detail tile nearby in the same column/row.

-The cobblestone dirt detail tiles can be used to create stone paths, also.

-The grass dirt detail tiles can help transition from grass to dirt if they hug the edge.

2 - Building walls

Wall tiles. These tiles are a complete 2-edge Wang Tileset.

The alternate tiles on the right are used to create the subtle suggestion of perspective as in the example. Not everyone likes the inconsistent projection. If you feel that way the tiles work fine without them.

NOTE:

If you plan to use the roof tiles above the walls, you will need to look ahead to make sure that your floor plan fits a possible roof configuration.

Horizontal roof canopy: Any width. Room heights of 4,7,10,13,16,..., etc.

Vertical roof canopy: Any height. Widths of 5,7,9,11,13,..., etc.

Vertical and horizontal sections can be combined to fit a variety of building sizes! See the roof tile section for examples.

3 - Fences, hedges

Fence and hedge tiles. These tiles are a complete 2-edge Wang Tilesets.

The alternate fence tiles are to be used as a pattern, not individually -- the highlighted green sides will work with with any of the other appropriate tiles, but the red sides will not.

4 - Interior floors

These tiles make up all the different interior floors.

Horizontal wood floor:

These (and the palette-swapped version) are the standard wood floor tiles. The left and right tiles can be used next to walls, while the two centre tiles can be repeated as need be.

Vertical wood floor:

Alternate wood floor. The left and right tiles are used next to walls. The tiles with the studs are designed to be used on alternate rows.

Brick/stone floor:

Stone/brick floor.

The 2-corner wang set on the right is used to transition to other ground tiles. Looks best with the dark detail tiles hugging corners of the room. Also, don’t be afraid to give the detail tiles room to breath with plenty of plain grey floor tiles.

Small interior step:

Another (almost) 2-corner wang set. This is used to create small step between any two ground tiles.

5 - Carpet / rug

These tiles are a nine-patch set (repeating sides, rather than stretching).

There are two centre tiles that you can use to design unique patterns for each rug.

6 - Doorway / Window / Wall grass detail

These tiles are all optional for creating doorways. Layer as in example. Windows/grass details will need to be drawn above wall tiles (or the corresponding roof-wall tile if using those tiles).

7 - Objects

Many of these objects are self-explanatory, so I’ll only cover a few of them.

Tree tiles. Alternate trunk for displaying on dirt tiles. Treetop is drawn above the trunk so that they may be hidden (if your game requires it).

The flowers are designed to be drawn on top of any grass ground tile. They look particularly good when used to detail the edge of a grass-dirt transition.

This 2-edge wang set can be used for store counters or display tables.

These mugs and candles are designed to be drawn above the table.

Unlike most of the other object tiles, these tiles are drawn to ‘hug’ a wall north of them. If you are using a strictly grid based movement system, you may consider allowing characters to enter these spaces.

To make these objects fill the space of the tile (like other objects), simply displace them down a few pixels.

8 - House roof

These tiles are used to create buildings with roofs displayed. Building roofs come in two orientations - horizontal or vertical. These two types can be combined to form a larger range of building shapes.

Horizontally oriented roof:

To maintain (near) accurate projection, try to keep the number of roof tiles below the ridge to be double the number above the ridge.

This means house heights of 4,7,10,13,16,...,etc. are OK.

Any house width possible.

(note: other house heights will fit seamlessly, but the house might look… a bit wonky, especially next to other houses with more correct projection).

Vertically Oriented Roof:

Any house height is possible. House widths of 5,7,9,11, etc. all possible. (for larger houses, the gable end tiles can be repeated as in the example)

Combining horizontal and vertical canopies:

These tiles make up the diagonal meeting of the two different roof types.

I find it easiest to start at the inside corners and working my way up until I find where the ridge finishes.

The diagonal tiles repeat as follows:

If the ridges meet at the same height, choose the appropriate ridge-joining tile, otherwise cap the ridge off with one of the four ridge-ending tiles.

Flush gable ends:

These tiles are used when a canopy finishes on the same row/column as the edge of another canopy. Easier to understand by looking at the examples! Great for L-shaped buildings.

9 - 2+ story buildings, staircase

These tiles can be used to create multi-storied buildings.

‘Stud’ tiles are optional, but add a little interest and help show the different floors.

The staircase/railing tiles can create multi-story sections for interiors.

For another example of these tiles in use, take a look at the inn/tavern in the example scene. I tried to see how far I could take these tiles in that scene, but I’m sure more is possible!
